# 蓝桥杯智能刷题与代码优化系统

基于大模型的算法竞赛学习平台，专为蓝桥杯等编程竞赛设计。

## 项目简介

这是一个集题目管理、在线评测、AI题解生成、代码优化建议于一体的智能刷题系统。

## 核心功能

- 📚 **题目管理**：支持蓝桥杯历年真题录入与分类
- ⚖️ **在线评测**：基于Docker的安全沙箱，支持Java/C++等语言
- 🤖 **AI智能题解**：**即将集成小米MiMo-V2.5模型**，利用100万超长上下文生成详细题解
- 💡 **代码优化建议**：AI自动分析代码，提供时间复杂度、空间复杂度优化建议

## 技术栈

- **后端框架**：Spring Boot 3.2.5
- **ORM框架**：MyBatis Plus 3.5.5
- **数据库**：MySQL 8.0
- **AI引擎**：小米MiMo-V2.5（即将集成）
- **构建工具**：Maven
